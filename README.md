MANUALLY INSTALL
----------------

1. Check out from GitHub

        git clone https://github.com/iamkc/vimrc.git ~/.vim

2. Install ~/.vimrc

        cd ~ && ln -s ~/.vim/vimrc ~/.vimrc 

3. Install plugin by Vundle, and lunch Vim 

        vim +BundleInstall
